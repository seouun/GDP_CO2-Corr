# 국가별 경제 성장과 이산화탄소 배출량 상관관계 분석

📌 국가별 GDP 성장과 이산화탄소(CO₂) 배출량 간의 구조적 관계를 시계열 데이터로 분석한 프로젝트  
(빅데이터분석과 시각화 과목 보고서 기반)


## 프로젝트 배경
- 산업혁명 이후 경제 성장은 전 세계적으로 에너지 소비와 탄소 배출을 증가시켜 기후 위기를 심화시킴
- 일부 고소득 국가는 **탈동조화(Decoupling)** 현상을 통해 경제 성장과 배출 감축을 동시에 달성
- 본 연구는 **국가별 GDP와 CO₂ 배출량 간의 상관관계**를 시계열 분석으로 파악하고,  
  인구·배출 수준·경제 수준 카테고리별 차이를 정량적으로 탐구함

  
## 🔹 데이터
- **출처**: Kaggle `gdp_co2_by_country_v2.csv` (World Bank 기반 가공 데이터)
- **기간**: 국가별 연도별 GDP, 인구, CO₂ 배출량 (2000~2020년대)
- **전처리**:
  - 결측치 제거 (특히 CO₂%와 GDP% 변화율)
  - 로그 변환(GDP Log, CO₂ Log)으로 분포 왜곡 보정
  - 국가별 특성 카테고리 부여 (인구 규모, 배출량 수준, GDP 단계)


## 🔹 방법론
1. **기초 상관관계 분석**
   - GDP 총량 vs CO₂ 총량
   - 1인당 GDP vs 1인당 CO₂
   - GDP 성장률 vs CO₂ 성장률
2. **국가별 사례 분석**
   - 강한 양의 상관관계 국가 (인도, 태국 등)
   - 강한 음의 상관관계 국가 (독일, 스웨덴 등 탈동조화 사례)
3. **카테고리별 분석**
   - 인구 규모별 (소규모 국가 vs 대규모 인구국)
   - 배출량 수준별 (Low ~ Very High)
   - GDP 수준별 (Low ~ High)


## 🔹 주요 결과
- 대부분의 국가는 여전히 **경제 성장과 CO₂ 배출이 강하게 양(+)의 상관관계**를 보임
- 일부 선진국(독일, 스웨덴, 영국)은 GDP 성장에도 불구하고 CO₂ 배출이 감소하는 **탈동조화 현상** 확인
- 인구 규모가 큰 국가일수록 GDP-CO₂ 상관관계가 더 강하게 나타남
- 고배출·고GDP 국가일수록 분포가 넓고, 일부는 음의 상관관계(감축)로 이동
- 정책적 시사점: **국가별 구조적 특성**을 고려한 맞춤형 기후 정책 필요
  

## 🔹 시각화 예시
- 국가별 GDP vs CO₂ 상관계수 히스토그램
- 카테고리별 박스플롯 (인구/배출량/GDP 단계별)
- 대표 국가(인도 vs 독일) 시계열 추이 비교 그래프

## 🔹 결론 및 시사점
- 여전히 다수 국가는 **“성장-배출 동조화”** 단계에 머물러 있음
- 그러나 일부 선진국은 정책·기술 혁신으로 **지속가능 성장 모델** 구현 중
- 기후 정의(Climate Justice) 관점에서, 단순 총량 비교가 아닌 **국가별 감축 역량과 구조적 차이**를 고려해야 함
